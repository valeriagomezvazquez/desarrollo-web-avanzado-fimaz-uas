# Práctica 2 – Parcial 1: HERENCIA Y REUTILIZACIÓN DE CÓDIGO EN PHP
## Objetivo
Implementar herencia mediante la extensión de clases, reutilizando atributos y métodos de una clase base
## Tecnologías utilizadas
- PHP 8+
- XAMPP
- PDO / MVC / REST (según aplique)
## Instrucciones de ejecución
1. Crear clase Usuario
2. Crear clase Admin
3. Crear Index
4. Colocar la carpeta del proyecto dentro del directorio `htdocs` de XAMPP.
5. Iniciar el servidor Apache desde XAMPP.
