# Práctica 3 – Parcial 1 SISTEMA DE USUARIOS CON VALIDACIONES Y EXCEPCIONES
## Objetivo
Desarrollar un sistema orientado a objetos más robusto,
integrando herencia, validaciones y manejo de excepciones para
simular un entorno profesional.
## Tecnologías utilizadas
- PHP 8+
- MySQL
- PDO / MVC / REST (según aplique)
## Instrucciones de ejecución
1. Crear las clases y subirlas al repositorio.
2. Colocar la carpeta `practica-3` dentro de `htdocs` en XAMPP.
3. Iniciar Apache desde el panel de control de XAMPP.
4. Abrir el navegador y acceder
